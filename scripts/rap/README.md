## About RAP

**RAP (Rapid Refresh)** is a NOAA continental-scale numerical weather prediction model.

- **Resolution**: 13 km
- **Coverage**: North America
- **Update Frequency**: Hourly
- **Forecast Length**: 0-21 hours
- **Best For**: Aviation forecasts, medium-resolution
